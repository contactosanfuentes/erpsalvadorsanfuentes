🛡️ Migración de insignias e imágenes a Supabase Storage

El sistema usaba 41 imágenes alojadas en imgur.com, que bloquea el acceso de forma errática (por eso algunas insignias no cargan). Este proceso descarga cada imagen desde tu navegador y la sube al almacenamiento propio del grupo (bucket payment-receipts/insignias (el mismo que ya usa tesorería para comprobantes)). Se ejecuta una sola vez; las que fallen se pueden reintentar.
